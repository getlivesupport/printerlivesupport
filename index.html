Common printer questions answered

Right-click your printer in Windows Settings, select 'See what's printing', click Printer menu and uncheck 'Use Printer Offline.' Restart both printer and router. On Mac, delete and re-add the printer.
Usually caused by stuck print queue, corrupted Print Spooler, or wrong default printer. Clear queue in Settings > Printers, restart Print Spooler via services.msc, and ensure printer is set as default.
Press WiFi button on printer, select your network, enter password. For HP use HP Smart app, Canon use Canon PRINT, Epson use Epson Smart Panel, Brother use Brother iPrint&Scan.
Visit your printer manufacturer's website using exact model number. Download the driver for your OS. Uninstall old driver first via Device Manager. Run installer and follow prompts. Always use official sources.
