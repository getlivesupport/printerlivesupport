Step-by-Step

How to Fix a Printer That Is Not Printing

Try these steps in order. If any step doesn't work, call (689) 336-4215 for live guided help.

  1. Check Printer Power & Status Lights

    Make sure the printer is powered on. No error lights should be blinking. Load paper in the tray and close all covers and doors fully.

  2. Clear the Print Queue

    Go to Settings → Devices → Printers. Click your printer, open the queue, and cancel all pending jobs. A stuck job blocks all new prints.

    Windows: Right-click the printer icon in the taskbar → Open queue → Cancel all

  3. Restart the Print Spooler Service

    Press Win + R, type services.msc, find "Print Spooler", right-click and select Restart. This clears software-level printer jams.

  4. Set as Default Printer

    In Settings → Printers, right-click your printer and select "Set as default printer." Disable "Let Windows manage my default printer" if it keeps switching.

  5. Reinstall the Printer Driver

    Uninstall the printer from Device Manager, then download the latest driver from the manufacturer's website using your exact model number. Reinstall fresh.

    HP: hp.com/support | Canon: usa.canon.com | Epson: epson.com/support | Brother: support.brother.com

  6. Call for Live Technician Help

    If none of the above steps resolved the issue, a certified technician can remotely diagnose and fix your printer in under 20 minutes.
